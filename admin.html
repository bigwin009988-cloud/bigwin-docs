📐 Banner字段+规范 Banner Specs

数据库字段 Database Fields

字段名类型必填默认值说明
idbigint✅auto主键自增
titlevarchar(50)✅—管理标题，仅后台可见
image_urlvarchar(500)✅—CDN图片URL
link_urlvarchar(500)❌null跳转链接
languageenum✅'all'all / en / hi
sort_orderint❌0越小越靠前
statusenum✅'draft'draft / active / expired
start_attimestamp❌null定时上架时间
end_attimestamp❌null定时下架时间
created_byFK→admin✅current创建人
created_attimestamp✅now()创建时间
updated_attimestamp✅now()最后修改时间
deleted_attimestamp❌null软删除时间

图片规范 Image Specs

属性要求
推荐尺寸750 × 320 px (比例 2.34:1)
最小尺寸640 × 270 px
最大文件500KB (上传时前端压缩)
格式JPG / PNG / WebP
上传处理① 压缩为WebP(quality:80) ② 生成缩略图200×85 ③ 上传CDN
校验失败尺寸不足→红色提示"图片尺寸不足" / 文件过大→"文件超过500KB"

前台展示规则 Frontend Display Rules

规则详情
轮播间隔默认3s自动切换（站点配置可调1-10s）
最大上架数同时最多8个。超过8个→提示警告但不阻止
语言匹配用户EN→显示language=en或all。用户HI→显示language=hi或all
排序sort_order ASC → created_at DESC
定时任务cron每分钟检查start_at/end_at，自动切换状态
无Banner前台隐藏轮播区域(不显示空白)
