1. V1 模块范围（6 个）

模块 主要页面/入口 是否做 执行口径
M1 登录/注册 /SignIn / /SignUp 做 支持 Phone+Password / Phone+OTP / Google；Password 输入提示使用 Password ( 6-16 digits )；切换文案使用 Sign In by OTP / Sign In by Password；OTP 临时 Mock：000000 成功，其他 6 位失败（本期不接短信平台）。
M2 Home /、/mobileAllGame 做 语言切换全站；Search 入口保持与竞品一致，输入框提示 Game name | Provider，提示文案 Search requires at least 3 characters；下拉刷新策略按 1goplus；未开放入口统一 Coming Soon。
M3 Daily Check-in 签到弹窗主链路 做 仅保留签到弹窗主链路（Quest 页不做）；结算时区为 UTC+5:30 00:00 (India) 且可配置；漏签后下次从 Day1 重新开始，不支持补签。
M4 coins 余额展示/资金回调相关 UI 做 回调未到账期间显示原先余额（不提前展示新余额）；到账后再刷新余额显示。
M5 游戏接入 /MobileIframe、游戏卡入口 做 点击未开放入口统一提示 Coming Soon；允许多开游戏；打开逻辑和竞品一致。
M6 个人中心 /AccountRight 做 仅允许修改昵称（接口 /api/account/md/name）；改密后当前设备保留登录并轮换 token，其他设备会话失效并要求重新登录。

5. API 对照（V1 必需）

接口 Method 鉴权 用途

/api/account/md/namePOST是修改昵称
/api/get/game/listPOST否首页游戏列表
/api/og/game/all/gamesPOST否All Games + Search
/api/home/res/load/home/group/configPOST否Home 分组配置
/api/home/res/load/home/bannersPOST否Home Banner

/api/get/fsdPOST是7 Day Gifts 状态
